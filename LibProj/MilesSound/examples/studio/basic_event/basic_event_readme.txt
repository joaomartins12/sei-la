!page(example_studio_basic_event, Basic Event)

_SDK Location: examples/studio/basic_event_

This demonstrates a very basic project that contains one Source and one
Event that plays that Source.

To recreate with your own sound:

$* Create a new project in Miles Studio
$* Select the *Bank* tab.
$* Drag a .wav file from your computer to the *Sources* list.
    A new source will be created under the bank *NewBank*.    
$* Right click on the new source, and select *Create Event To Play*.


Upon completion, there will be an event listed under *Events*
that will play your new sound once compression completes. To
test the event, select it in the *Events* list and press Spacebar.
